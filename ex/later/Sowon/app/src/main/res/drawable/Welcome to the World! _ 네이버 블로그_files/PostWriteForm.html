object 키워드: 클래스 선언과 인스턴스 생성

위치이동제목 배경 사진 삭제
취소확인

  • object 키워드: 클래스를 정의하면서 동시에 인스턴스(객체)를 생성한다

    • 객체 선언(object declaration): 싱글터는 정의하는 방법. 인스턴스가 하나만 필요한 클래스일 경우 object로 선언하고 생성한다. 객체 선언은 클래스 선언과 그 클래스에 속한 단일 인스턴스의 선언을 합친 선언이다. 클래스와 마찬가지로 객체 선언 안에도 프로퍼티, 메서드, 초기화 블록 등이 들어갈 수 있지만, 생성자는 객체 선언에 쓸 수 없다. 객체 선언에 사용한 이름 뒤에 마침표를 붙이면 객체에 속한 프로퍼티에 접근할 수 있다.
